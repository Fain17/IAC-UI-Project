# iac_ui_agent Frontend

A modern React + TypeScript frontend using Vite.

## Directory Structure
```
frontend/
  ├── src/
  │   ├── api.ts
  │   ├── components/
  │   ├── pages/
  │   ├── styles/
  │   └── ...
  ├── public/ (optional, usually empty)
  ├── index.html
  ├── package.json
  ├── tsconfig.json
  ├── vite.config.js
  └── ...
```

## Usage
```bash
npm install
npm run dev
```

- Home, Login, Register, and Settings pages included
- JWT authentication and protected routes
